2.¿Qué es el modelo Box en HTML?

  1. Es el contenido central de todo elemento
  2. Es el borde de la línea que rodea la caja como la frontera que rodea al elemento
  3. El área donde se muestra el contenido, cuyo tamaño puede cambiarse utilizando propiedades como width y height
  4. La respuesta correcta es la 1 y la 3

3. ¿Qué aspectos importantes tiene el modelo de caja en CSS?

  1. Incluye sus rellenos y márgenes, que son generadas para los elementos y que se disponen de acuerdo al modelo de formato visual
  2. Se utiliza para dibujar un contorno alrededor de un elemento
  3. El relleno, el margen y el borde mantienen a los otros elementos alejados de la caja
  4. Está definido por los valores de propiedad display, como block y inline

4. ¿Qué es el modelo de cajas box model en CSS?

  1. Es una lista, que se presenta usando display: flex
  2. Es el comportamiento de CSS que hace que todos los elementos de las páginas se representen mediante cajas rectangulares
  3. Es un espacio invisible que hay alrededor de la caja
  4. Es un punto clave a la hora de entender los márgenes es el concepto de colapso del margen
